전체 선택자: * {css 코드}

전체 선택자는 모든 요소를 한 번에 선택할 수 있습니다.

전체 선택자 이후 태그 선택자로 태그별로 별개의 스타일 적용이 가능합니다.

아이디 선택자: #id 속성값 {css 코드}

클래스 선택자: .class 속성값 {css 코드}

기본 속성 선택자: a[href] {css 코드} a 태그의 target 속성값이 _blank인 요소에 적용: a[target="_blank"]

그룹 선택자: h2, h3 {css 코드}

하위 선택자: 선택자1(div) 선택자2(p) {css 코드}

p태그 내용1

p태그 내용2

인접 형제 선택자: 이전 선택자 + 대상 전택자 {css 코드}

인접 형태 선택자의 대상 선택자: h5
한 번도 방문하지 않은 링크일 때: a:link {css 코드}, 방문한 링크일 때: a:visited {css 코드}"

요소에 마우스를 올리면 해당 태그가 선택자로 지정: p:hover {css 코드}

요소를 마우스로 클릭하고 있는 동안 해당 태그가 선택자로 지정: p:active {css 코드}
